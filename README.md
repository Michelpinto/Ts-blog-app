## Features

1. Create account with email and password
2. Create a blog with rich text
3. See your past blog posts
4. delete blogs

# Tech stack

1. Typescript
2. Nodejs/express
3. MongoDB
4. Redux toolkit
5. axios
6. styled-components
7. cypress testing
